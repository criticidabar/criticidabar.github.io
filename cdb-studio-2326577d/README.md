# CdB Studio Mobile v0.2

Web app statica mobile-first per creare caroselli Critici da Bar.

## Novità

- accesso con PIN locale al primo avvio;
- pagina marcata `noindex`;
- funzionamento da sottocartella GitHub Pages;
- pinch e maniglia sul testo modificano la dimensione reale e rifanno gli a capo;
- pinch su immagine e logo modifica la scala;
- comando **Adatta testo**;
- autosave locale, export PNG/SVG/ZIP e progetto JSON.

## Limite della protezione

Il PIN è una barriera locale e intenzionalmente semplice. Il path casuale e il `noindex` evitano l’accesso casuale, ma non sostituiscono un’autenticazione server. I progetti non vengono caricati nel repository: restano nel browser.
