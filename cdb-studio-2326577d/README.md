# CdB Studio v0.18

Aggiornamento desktop: trascinamento più naturale, eliminazione completa, scorciatoie da tastiera e copia/incolla interno degli elementi.

## Scorciatoie
- Ctrl/Cmd + Z: annulla
- Ctrl/Cmd + Shift + Z oppure Ctrl + Y: ripeti
- Ctrl/Cmd + C / V: copia e incolla elemento
- Ctrl/Cmd + X: taglia elemento
- Ctrl/Cmd + D: duplica
- Canc/Delete o Backspace: rimuove l’elemento selezionato
- Frecce: sposta di 1 px
- Shift + frecce: sposta di 10 px
- Durante il trascinamento: Shift blocca un asse, Alt disattiva temporaneamente le guide

Gli elementi strutturali (titolo, testo, logo, numero) vengono nascosti invece di essere distrutti e possono essere riattivati dal pannello Livelli.
